# Camel Up Simulator 🐪

A digital simulation of the board game *Camel Up*, modeling probabilistic movement, betting, and race outcomes.

## What it does
- Simulates camel movement using dice-based probability
- Tracks positions, stacking mechanics, and race state
- Computes expected value of bets
- Supports Monte Carlo simulations for strategy analysis

## Why I built it
Jane Street AMP Project!

## Technologies
- Python
- NumPy
- Monte Carlo simulation
